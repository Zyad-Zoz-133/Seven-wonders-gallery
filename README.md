# 🌍 Seven Wonders Gallery

A responsive website showcasing the **Seven Wonders of the Ancient World** and the **New Seven Wonders of the World**. The project focuses on building a clean, modern gallery layout while practicing Bootstrap's Grid System and responsive web design.

## 📖 About the Project

Seven Wonders Gallery is a simple educational website that allows users to explore both the ancient and modern wonders of the world. Each wonder is displayed inside a responsive card with an image, location, short description, and a "Learn More" button.

This project was built to improve my understanding of responsive layouts using Bootstrap and modern CSS.

---

## ✨ Features

- 🏛️ Ancient Seven Wonders section
- 🌍 New Seven Wonders section
- 📱 Fully responsive design
- 🎨 Modern dark theme
- 🖼️ Responsive image cards
- 📍 City and country information
- 🔗 Navigation bar with smooth scrolling
